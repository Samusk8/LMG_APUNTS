¿Qué es HTML?

Html (HyperText Markup Language) es el lenguaje de marcass para crear páginas web. Define la estructura de la página con etiquetas que el navegador interpreta para mostrar contenido.

Breve Historia

Creat per Tim Berners-Lee el 1991.
HTML 2.0 (1995): Primer estàndard oficial.
HTML5 (2014): Última versió amb noves funcionalitats com suport per a vídeo, àudio i gràfics.
