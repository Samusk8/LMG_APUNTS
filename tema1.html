¿Qué es HTML?

Html (HyperText Markup Language) es el lenguaje de marcas para crear páginas web. Define la estructura de la página con etiquetas que el navegador interpreta para mostrar contenido.

Breve Historia

Creat per Tim Berners-Lee el 1991.
HTML 2.0 (1995): Primer estàndard oficial.
HTML5 (2014): Última versió amb noves funcionalitats com suport per a vídeo, àudio i gràfics.

Estructura básica de un documento HTML

Un documento Html se basa en una estructura jerárquica de etiquetas que definen las diferentes partes de una página.

Cada página Html sigue una estructura similar, Comienza por !DOCTYPE , seguido de las etiquetas html, head/body ...

< !doctype html>

  < html>
           < head>< /head>

           < body>< /body>
  < /html>

Etiquetas Básicas

Title

Define el título que aparece en la pestaña del navegador web

Títulos (h1-h6)

Se utiliza para definir títulos y subtítulos en la página

Párrafo (p)

Se utiliza para definir bloques de texto o párrafos.

Intro (br)

Fuerza el texto a comenzar en una nueva línea

Espacios (& nbsp;)

Inserta un espacio en blanco que no se separa entre líneas

Anidamiento

Diferencias entre etiquetas y elementos

Etiquetas (tag):

Es la parte que define el inicio y el final de un componente Html Se delimita con los símbolos <>

Elementos

Un elemento incluye una etiqueta de abertura, el contenido y la etiqueta de cierre.
Los elementos pueden estar vacios o tener otros elementos dentro.

Elementos anidados

El anidamiento se refiere a colocar un elemento HTML dentro de otro.

Es importante seguir correctamente el orden de las etiquetas: La primera que se abre es la ultima que se cierra.

< !doctype html>

  < html>
            <head>
                        <title></title>
            </head>

            <body>
                        <div>
                                    <p></p>
                        </div>
            </body>
  < /html>

Comentarios HTML

  • Los comentarios enn html son lineas de texto que se añaden al codigo para dejar notas o explicaciones, pero no son visibles para los usuarios den la página web.
  • Se utilizan para ayudar a los desarrolladores a documentar el codigo, describir secciones o dejar indicaciones.
  • Los comentarios no afectan el rendimiento ni visualización.

Atención

Bones Pràctiques amb Comentaris:

  • Mantenir-los clars i concisos: Els comentaris han de ser breus i proporcionar informació útil.
  • Evitar comentaris excessius: Només afegeix comentaris on sigui necessari per evitar sobrecarregar el codi.
  • No deixar comentaris innecessaris en producció: Els comentaris en entorns de desenvolupament poden ser útils, però en codi de producció és millor deixar només els essencials.

Imagenes

Introducir Imagenes

  • L'etiqueta <img> s'utilitza per inserir imatges en una pàgina web.
  • No té etiqueta de tancament, ja que és un element auto-tancat.
  • És essencial proporcionar informació sobre la imatge perquè es mostri correctament i per millorar l'accessibilitat.

Atributos:

  1. src (source)
    • Defineix la font o el camí de la imatge.
    • Pot ser un camí relatiu o una URL completa.
  2. alt (alternate text)
    • Proporciona un text alternatiu que es mostra si la imatge no es carrega o és llegida per lectors de pantalla.
    • És fonamental per a la accessibilitat i SEO.
  3. width i height
    • Especifica l'amplada (width) i l'alçada (height) de la imatge en píxels o percentatges.
    • És important per controlar la mida de la imatge a la pàgina.

Atención

Bones Pràctiques:

  • Sempre utilitza el atribut alt per garantir una bona accessibilitat i SEO.
  • Evita inserir imatges molt grans sense reduir la seva mida, ja que això pot afectar el rendiment de la pàgina.
  • Si les imatges són responsives (se adapten a diferents dispositius), utilitza percentatges en lloc de píxels per width i height.

Enlaces

Introducir enlaces

  • L'etiqueta <a> s'utilitza per crear enllaços (links) a altres pàgines web, documents, seccions dins de la mateixa pàgina o qualsevol altre recurs.
  • El element essencial per als enllaços és el atribut href que defineix la URL a la qual redirigeix el usuari.
  • El text entre la etiqueta d'obertura i la de tancament és el que els usuaris veuran i podran clicar per ser redirigits.

Atributos:

  1. href (hypertext reference):
    • Defineix la URL de destinació de l'enllaç.
    • Pot ser un enllaç absolut (una URL completa) o relatiu (camí dins del mateix domini).
  2. target:
    • Defineix on s’obrirà l’enllaç: a la mateixa pestanya, en una nova pestanya o en un nou marc.
    • Els valors més habituals són:
      • _blank: Obre l'enllaç en una nova pestanya.
      • _self: Obre l'enllaç a la mateixa pestanya (valor predeterminat).

Atención

Bones Pràctiques:

  • Utilitza enllaços descriptius en lloc de genèrics com "Clica aquí". Això millora l’accessibilitat i el SEO.
  • Utilitza target="_blank" només quan sigui realment necessari, ja que obrir enllaços en noves pestanyes pot confondre alguns usuaris.
  • Sempre assegura’t que els enllaços siguin accessibles afegint una descripció adequada per a usuaris amb lectors de pantalla.

Listas en HTML

Llistes No Ordenades (<ul>):

  • S'utilitzen per a llistes d’elements sense un ordre específic.
  • Els elements es mostren amb vinyetes o altres símbols abans de cada ítem.
  • Cada element de la llista es defineix amb l’etiqueta
    • <li> (list item).

Llistes Ordenades (<ol>):

  • S’utilitzen per a llistes en què l'ordre dels elements és rellevant.
  • Els elements es numeren automàticament (1, 2, 3, etc.).
  • Cada element de la llista es defineix amb l’etiqueta
    • <li> (list item).

Atributos

  1. Atribut type
    • Se utiliza para definir el tipo de numeración
      • 1
      • A
      • a
      • I
  2. Atribut Start
    • Define el numero o letra para comenzar la numeración

Atención

Bones Pràctiques:

  • Utilitza llistes no ordenades (<ul>) quan l'ordre dels ítems no sigui important.
  • Utilitza llistes ordenades (<ol>) quan l'ordre sigui rellevant.
  • Personalitza les llistes segons les necessitats visuals o estilístiques utilitzant els atributs adequats.

Tablas básicas HTML

Etiqueta <table>:

L’etiqueta <table> defineix la taula i actua com el contenidor principal.

Etiqueta <tr> (table row):

Defineix una fila dins de la taula.

Etiqueta <td> (table data):

Crea una cel·la dins d’una fila de la taula (dades de la taulla)

Etiqueta <thead> (table head):

Defineix l’encapçalament de la taula, agrupant les files d'encapçalament (habitualment les que contenen <th>).

Etiqueta <tbody> (table body):

Agrupa el cos de la taula, és a dir, les files de dades. Se sol fer servir per separar les dades dels encapçalaments.

Etiqueta <th> (table header):

S’utilitza per crear una cel·la d'encapçalament en una taula. El text dins de <th> apareix en negreta i centrat per defecte.

Atributs de Taula:

  • border: Afegeix una vora a la taula i a les cel·les
  • cellpadding:Defineix l’espai entre el contingut de la cel·la i la vora de la cel·la.
  • cellspacing:Controla l'espai entre les cel·les de la taula.

Atención

Bones Pràctiques:

  • Utilitza <thead> i <tbody> per organitzar les taules, especialment en taules més grans, per tal de separar clarament l’encapçalament del cos de la taula.
  • Mantingues les taules simples per a dades tabulades i utilitza CSS per al disseny més avançat de les taules (colors, mida de la font, etc.).
