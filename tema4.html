Selectores avanzados

Els selectors avançats en CSS permeten aplicar estils amb molta més precisió i flexibilitat, permetent seleccionar elements específics en funció de la seva posició, relació amb altres elements o atributs.

Selectors d'atribut

Els selectors d'atribut permeten aplicar estils als elements HTML que contenen un determinat atribut o valor d'atribut. Són especialment útils per estilitzar elements de formularis o en casos en què utilitzis atributs personalitzats.

input[type="text"] {
border: 1px solid blue;
}

a[href*="example"] {
color: green;
}

img[src^="https"] {
border: 2px solid black;
}

Selectors combinats

Els selectors combinats permeten seleccionar elements que tenen una relació específica amb altres elements dins del document. Els combinadors més comuns són:

  1. Combinador de descendents
  2. Combinador de fills directes
  3. Combinador de germans adjacents
  4. Combinador de germans generals

div p { color: red; }

div > p { color: blue; }

h1 + p { font-weight: bold; }

h2 ~ p { color: gray; }

3. Selectors pseudo-classe

Les pseudo-classes són selectors que s'utilitzen per aplicar estils a un element en un estat o condició particular. Això permet canviar l'estil d'un element quan l'usuari hi interactua o quan l'element compleix certs criteris.

  1. :hover: Aplica estils quan l'usuari passa el ratolí per sobre de l'element.
  2. :nth-child(): Selecciona elements en funció de la seva posició entre els seus germans.
  3. :first-of-type: Selecciona el primer element del seu tipus dins del seu pare.
  4. :not(): Selecciona tots els elements excepte aquells que coincideixin amb el selector especificat.

a:hover { color: red; }

tr:nth-child(odd) { background-color: #f0f0f0; }

p:first-of-type { font-weight: bold; }

div:not(.important) { background-color: yellow; }
